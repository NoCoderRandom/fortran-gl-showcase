# Future Scene Ideas

Ideas saved for later continuation.

## Array-Focused Scene Follow-Ups

- Matrix Ripple Cathedral
  An array-driven interference field where multiple wave sources and neighbor operations build vaulted ripple structures, then OpenGL shades the result like a luminous cathedral floor or ceiling.

- Tensor Heatmap Canyon
  A layered field built from stacked array transforms, gradients, and smoothing passes that reads like a canyon or topographic survey with cinematic lighting.

- Spectral Array Waterfall
  A flowing field where array-based advection, banding, and temporal phase shifts create cascading ribbons of color that feel like a data waterfall.

## General Direction

- Keep the scene meaningfully tied to modern Fortran strengths:
  whole-array math, `spread`, `cshift`, reductions, and field updates on dense numeric grids.

- Use OpenGL mainly as the presentation layer:
  shading, lighting, bloom, color grading, and camera treatment on top of data produced in Fortran.

- Favor categories that look visually distinct from the existing fractal, particle, and shader-only scenes.
